# Customer Support Ticket Classification

## Overview
This project classifies customer support tickets into categories and assigns priority levels using machine learning.

## Tools Used
- Python
- Scikit-learn
- Google Colab

## Features
- Text classification (Billing, Technical, General)
- Priority prediction (High, Medium, Low)
- Multiple ticket testing

## Output Example
Input: Payment failed  
Category: Billing  
Priority: High  

## Conclusion
This system helps support teams respond faster by automatically classifying tickets.
